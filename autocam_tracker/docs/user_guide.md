# AutoCam Tracker - 操作指南 (User Guide)

歡迎使用 AutoCam Tracker。本系統專為多機位、高密度的車流追蹤所設計，具備強大的 ReID 特徵記憶庫與跨機位自動學習功能。

---

## 1. 基礎播放控制
在畫面正下方，您可以使用以下控制選項：
* **空白鍵 (Spacebar)**：隨時暫停 / 繼續播放影片。
* **進度條 (Seeker)**：拖曳以快速跳轉至影片的特定時間點。
* **播放速度 (Speed)**：可選擇 0.5x, 1x, 2x 來放慢或加快影片播放速度。
* **Reset DB**：一鍵清除所有目前記憶的車輛特徵與追蹤狀態，重新開始。
* **Reset Crop**：取消目前的畫面錨定，將視野恢復為全域預覽。

---

## 2. 車輛追蹤與特徵綁定 (核心操作)

在左上方的主要預覽畫面中，您可以透過滑鼠與鍵盤的組合，精準控制系統要記住哪些車輛：

### 🎯 單純觀察 (無記憶)
* **`左鍵點擊 (Left Click)`**
  * 僅將右側的「裁切視角 (Crop View)」對焦到該車輛上。
  * 系統**不會**將這台車存入資料庫，如果車輛被擋住或切換機位，系統就會忘記它。

### ➕ 建立新目標 (New GID)
* **`Shift + 左鍵點擊`**
  * 告訴系統：「這是一台重要目標，請幫它建立一個專屬編號 (例如 G1)」。
  * 系統會立刻擷取這台車現在的長相特徵存入資料庫。

### 🔄 補充特徵 (Add Feature)
* **`Ctrl + 左鍵點擊`**
  * 當系統正在追蹤某台車時，如果您覺得它轉彎了、變亮了，或是長得不太一樣了，可以按著 Ctrl 再點它一次。
  * 系統會把這個「新角度的長相」加進該目標的記憶庫中。記憶越多角度，系統追蹤得越死！

### 🖱️ 快速選單
* **`右鍵點擊 (Right Click)`**
  * 點擊車輛會跳出快捷選單，您可以選擇 `[+] Bind as New Target` 來建立新標籤，或是選擇畫面上已經存在的 G1, G2 等標籤，直接把特徵加進去。

> **💡 專業提示 (暫停操作法)**：
> 當車流非常密集、車速很快時，您可以**按下空白鍵暫停影片**。在暫停狀態下，您依然可以悠哉地用 `Shift+點擊` 或 `Ctrl+點擊` 來為多台車輛綁定特徵。系統會即時寫入資料庫並在左下角顯示更新！

---

## 3. Anchor DB (特徵資料庫管理)

位於介面左下角的列表是系統的「大腦記憶庫」，它會顯示系統目前追蹤了哪些車、以及對它們有多熟悉。

### 介面解讀
* **`car G1 (3 Feat)`**：代表系統給這台車編號為 1，並且總共為它記憶了 3 種不同的長相特徵。
  * **`Shot 1 [2]`**：在這台車處於「第一台攝影機 (Shot 1)」時，記錄了 2 個特徵。
  * **`Shot 2 [1]`**：當畫面切到「第二台攝影機 (Shot 2)」時，記錄了 1 個特徵。

### 功能與自動學習
* **強制鎖定**：如果您想要重新看 G1，只要在列表中**左鍵點擊 `car G1`**，鏡頭就會立刻切換回去找它。
* **跨機位自動學習**：當影片突然切換到不同的監視器角度 (Camera Cut) 時，只要系統認出了 G1，它就會**自動幫您把 G1 在新鏡頭的樣子拍下來存進去**。您會看到列表全自動長出 `Shot 2`、`Shot 3` 等分支，完全不需要您手動操作！
